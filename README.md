## 🛠 Study Procedure

### Before Study

-   발표자는 발표자료를 해당주차 폴더에 업로드

-   다른 스터디원들의 질문들을 보고 답변하기 (Option)
    -   질문에 대한 답변은 Pr로만 올려두기

### Study

-   발표자가 학습내용 발표 -> Q&A 시간

-   스터디 요약 각자 폴더에 작성
-   발표자가 pr 정리하기
